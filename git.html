GIT NOTE FOR PROFESSIONAL

    Thiết lập Username và Email

  • git config --global user.name "Pham Son
  • git config --global user.email "sondeptrai688@gmail.com"

    • Dữ liệu trong Git có 3 trạng thái

  • commited: Dữ liệu đã được lưu trữ an toán trong DataBase
  • modified: Dữ liệu trong file có thay đổi nhưng chưa lưu vào DataBase
  • staged: Dữ liệu đã được đánh dấu và chuẩn bị commited

    • modified => staged => commited

    Tạo ra thư mục lưu trữ project:

  • mkdir Filename (mkdir MyCode)

    • Vào thư mục Filename bằng lệnh:

  • cd Filename/ (cd MyCode/)

    • Tạo file:

  • touch filename (index.html)

    • Sửa đổi code trong filename:

  • code filename (code index.html)

    • Biến Filename thành một kho chứa git:

  • git init

    • Khởi tạo phiên bản commited

  • git commit -m "First Commit"

    • Thể hiện trạng thái của Project:

  • git status

    • Thể hiện các phiên bản commit

  • git log --online

    • So sánh giữa 2 commit:

  • git diff

    • So sánh commited cuối với staged:

  • git diff --staged

    • Phục hồi file về commit bất kỳ:

  • git checkout id commit1 . hoặc filename
